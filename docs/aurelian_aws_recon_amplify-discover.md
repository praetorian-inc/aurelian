## aurelian aws recon amplify-discover

Discovers Amplify app branches by probing CloudFront distribution IDs against the <branch>.<distribution-id>.amplifyapp.com URL pattern. Amplify apps and their backing CloudFront distributions share the same distribution ID.

